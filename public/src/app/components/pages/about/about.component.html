This fullstack project uses the following technologies:

  • MongoDB
  • Express
  • Angular
  • Node
  • Bootstrap
  • Deployed with AWS EC2

This project began by following the tutorial by Brad Traversy at https://www.youtube.com/watch?v=Fdf5aTYRW0E&t=2390s. Originally, this projects connected to the placeholder JSON as https://jsonplaceholder.typicode.com/todos. I removed the dummy data and added Express and MongoDB to make this application full stack MEAN. I have also added bootstrap and edited the CSS for stylepoints. The github for this project is at https://github.com/SalvatoreSantamaria/angulartodolistfullstack.
